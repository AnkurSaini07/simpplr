
## ENV Configurations
* HOST
* PORT
* YELP_API_KEY
* YELP_CLIENT_ID

## APIS
* /shops : give list of top 10 ice cream shop sort by rating.
* /reviews/:shopId : give list reviews by shopId.
* /shopsWithReviews : give list of top 10 ice cream shop sort by rating with reviews.
